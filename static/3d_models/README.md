# النماذج ثلاثية الأبعاد للمواقع المقدسة

هذا المجلد يحتوي على النماذج ثلاثية الأبعاد للمواقع المقدسة المختلفة.

## الصيغ المدعومة:

### 1. GLB (GL Transmission Format Binary)
- **الاستخدام**: العرض على الويب
- **المميزات**: جودة عالية، تحميل سريع
- **الحجم**: 37-50 MB
- **دقة النسيج**: 1K-2K

### 2. glTF (GL Transmission Format)
- **الاستخدام**: التطوير والتحرير
- **المميزات**: قابل للقراءة، مرن
- **الحجم**: 24 MB
- **دقة النسيج**: 1K

### 3. USDZ (Universal Scene Description)
- **الاستخدام**: الواقع المعزز على iOS
- **المميزات**: محسن لـ AR
- **الحجم**: 13 MB
- **التوافق**: iOS 12+

### 4. FBX (Filmbox)
- **الاستخدام**: الصيغة الأصلية
- **المميزات**: جودة كاملة، جميع التفاصيل
- **الحجم**: 87 MB
- **الاستخدام**: التطوير والتحرير المتقدم

## النماذج المتاحة:

### الكعبة المشرفة (kaaba)
- `kaaba.glb` - للعرض على الويب
- `kaaba.gltf` - للتطوير
- `kaaba.usdz` - للواقع المعزز iOS
- `kaaba.fbx` - الصيغة الأصلية

### المسجد الحرام (masjid-al-haram)
- `masjid-al-haram.glb` - للعرض على الويب
- `masjid-al-haram.gltf` - للتطوير
- `masjid-al-haram.usdz` - للواقع المعزز iOS
- `masjid-al-haram.fbx` - الصيغة الأصلية

## إرشادات الاستخدام:

1. **للعرض على الويب**: استخدم GLB
2. **للواقع المعزز**: استخدم USDZ (iOS) أو GLB (Android)
3. **للتطوير**: استخدم glTF أو FBX
4. **للتحميل**: جميع الصيغ متاحة

## متطلبات النظام:

- **المتصفح**: Chrome 67+, Firefox 60+, Safari 12+
- **الواقع المعزز**: iOS 12+ أو Android 7+
- **WebGL**: 2.0 مطلوب
- **الذاكرة**: 2GB RAM على الأقل

هذه النماذج مخصصة للاستخدام التعليمي والديني فقط.
